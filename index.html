Stacks

만들고자 하는 목표에 대해 필요한 기술들을 빠르게 습득하고 구현합니다. 할 수 있는 경계선을 더 넓혀 나가고자 합니다.
기술 구분 로고 활용 수준 평가
Java 객체지향의 개념과 장점을 이해하고 활용하고 있습니다. 또한 반복되는 코드를 줄이기 위해 지속적으로 리팩토링을 진행하며 함수, 클래스, 인터페이스를 활용함과 동시에 오버로딩, 오버라이딩을 같이 사용함으로써 조금 더 편한 코딩을 하기 위해 꾸준히 노력하고 있습니다. 단순히 기술만 쓰는 것이 아니라 어떤 원리로 기술이 작동하는지 이해하고자 공부하고 있습니다.
OracleDB SubQuery, Join을 활용하여 많은 양의 데이터를 보유하고 있는 테이블에서 원하는 형태의 데이터를 추출할 수 있습니다. 또한, 프로시저와 함수, 트리거를 활용하여 Select를 더욱 편하게 사용할 수 있습니다. Index를 사용하여 매우 많은 양의 데이터에 빠르게 접근할 수 있습니다.
HTML HTML에서 자주 사용하게 되는 태그를 잘 사용하고 있습니다. 하나의 태그에 꽂혀 그것에만 매달리지 않고, 코딩을 더 효율적으로 할 수 있는 태그가 새로 생긴다면 그것에 대해 공부하며 더 발전된 실력을 보유하기 위해 노력하고 있습니다.
CSS HTML의 서식을 꾸며주는 CSS의 다양한 태그들을 사용하여 보기 좋은 웹 페이지를 만들 수 있도록 노력하고 있습니다. 과거에 주로 사용하던 Style 태그 뿐만 아니라 최신 태그들도 공부하며 더 효율적으로 코딩하기 위해 항상 배우는 자세를 유지하고 있습니다.
Javascript 지속적으로 Javascript의 기술들을 습득 중입니다. 특히, 반복되는 코드를 어떻게 하면 줄일 수 있을지에 초점을 맞춰 끊임없이 고민하고 있으며 Function과 Foreach문을 통해 비슷한 동작을 수행하는 코드들을 모아서 반복되는 작업을 줄일 수 있도록 노력하고 있습니다.
객관화하기 어려운 관계로, 제가 이해하고 구현할 수 있는 수준을 작성했습니다.

Front End

HTML, CSS, Javascript를 배우면서 어느 하나에 치우치지 않기 위해 골고루 배우려고 노력하고 있습니다. 순수 언어를 통한 코딩 뿐만 아니라 Bootstrap을 활용하여 코딩 시간을 단축하고 있습니다.

Back end

Java를 가장 많이 사용하고 있습니다. 하지만 Oracle DB도 같이 학습함으로써 다량의 데이터를 다루고 그것을 토대로 유의미한 결과를 얻을 수 있도록 Java와 연동하여 원활하게 사용하기 위한 방법을 끊임없이 고민하고 있습니다.

Data base

Oracle DB를 사용하며 클라이언트로 SQL Developer를 같이 사용하고 있습니다. 쉬운 데이터 뿐만 아니라 복잡하고 의미있는 데이터를 어떻게 하면 얻을 수 있을지 항상 고민하여 연습하고 있습니다.

Utilities

아직 많은 유틸리티를 다루지는 못합니다. 하지만 차근차근 단계를 밟아 나가며 필요하다고 생각되는 유틸리티가 있다면 인터넷과 책을 병행하며 학습하기 위해 노력하고 있습니다. 최근에는 Github를 다루는 방법을 공부했고, 보다 쉽게 다른 사람의 파일 혹은 제 파일을 원활하게 공유하는 즐거움을 깨닫게 되었습니다.
